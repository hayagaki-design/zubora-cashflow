# ずぼらキャッシュフロー

スマホで金額だけ入れて、今月の入金・出金・残りをすぐ見るためのミニアプリです。

## 使い方

1. このフォルダでローカルサーバーを起動します。

   ```powershell
   python -m http.server 4173 --bind 127.0.0.1
   ```

2. ブラウザで開きます。

   ```text
   http://127.0.0.1:4173/
   ```

## メモ

- データはブラウザ内に保存されます。
- 右上のボタンからCSVを書き出せます。
- `analyze.html` で、スマホから書き出したCSVをPCに読み込んで集計できます。
- スマホのホーム画面に追加すると、かなりアプリっぽく使えます。

## ワンタッチ同期

Googleスプレッドシートに同期する場合:

1. Googleスプレッドシートを新規作成します。
2. `拡張機能` → `Apps Script` を開きます。
3. `google-apps-script.gs` の中身を貼り付けます。
4. `デプロイ` → `新しいデプロイ` → `ウェブアプリ` を選びます。
5. 実行ユーザーは自分、アクセスできるユーザーは `全員` を選びます。
6. 発行されたURLをスマホ側の `同期先を設定` に貼ります。
7. 以後は `同期` ボタンでスプレッドシートに送れます。

同期URLを知っている人は送信できるので、URLは外に出さないでください。

## 公開する

静的ファイルだけなので、GitHub Pages / Netlify / Vercel などで公開できます。
公開先はHTTPSになるため、ホーム画面追加とオフライン起動にも対応します。
