# FULL.

スマホ向けの食事記録PWAです。食べたものを文章で入力し、「軽め・普通・がっつり・多すぎ」から食べた感を選んで記録します。

## ローカルで試す

```powershell
python -m http.server 4174 --bind 127.0.0.1 --directory eat-log
```

ブラウザで `http://127.0.0.1:4174/` を開きます。データはブラウザのローカルストレージに保存されます。

## 仕様

- 「と」「、」「,」「・」「それから」「あと」で食品を自動分割
- 午前4時を1日の境界として集計
- 初期の食べすぎラインは9点（5〜15点で変更可能）
- 履歴の編集・削除
- 独自コース「小回り」「大回り」を1〜99周で記録
- 運動履歴の集計・削除
- Googleスプレッドシートへの同期（Apps Script URLを設定）
- オフライン起動対応

## Googleスプレッドシート同期

1. `食事ログ` と `運動ログ` シートを持つスプレッドシートを用意します。
2. スプレッドシートで「拡張機能」→「Apps Script」を開きます。
3. `google-apps-script.gs` の内容を貼り付けて保存します。
4. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」を選びます。
5. 実行するユーザーを自分、アクセスできるユーザーを全員にしてデプロイします。
6. 発行された `/exec` で終わるURLを、アプリの「設定」→「スプレッドシート同期」に保存します。

同期URLはGitHub Pagesには含まれず、設定した端末のローカルストレージだけに保存されます。
